Συνήθη προβλήματα
1 Το ΣΥ.ΔΕ.ΤΑ εκτύπωνεται χωρίς στοιχεία Αποστολέα Αυτό συμβαίνει όταν τα ΣΥ.ΔΕ.ΤΑ παράγονται μέσω Web Services και εκτυπώνονται απο το πρόγραμμα του πελάτη
2 Κατα την κλήση του Web Services μου βγάζει Λάθος Κωδικός 1. Βεβαιωθείτε ότι οι κωδικοί που βάζετε είναι ορθοί (βάσει των ορισμών τους σε κάθε Web Service)
2. Βεβαιωθείτε ότι ορίζεται σωστά τους κωδικούς πρόσβασης για την κλήση του WS. Για παράδειγμα έστω ότι σας έχουν δοθεί τα κάτωθι στοιχεία: κωδικό πελάτη 010101 0005, κωδικό χρήστη 0011111 και password = a11111 και θέλετε να ‘καλέσετε’ το CREATEAWB.WSDL, η αντιστοιχία των κωδικών είναι: PEL-USER-CODE ='0011111', PEL-USER-PASS='a11111' , PEL-APOST-CODE = '010101' και PEL-APOST-SUB-CODE = '0005'
3. Βεβαιωθείτε ότι οι κωδικοί που βάζετε κατα την κλήση της εκτύπωης είναι o master (π.χ έστω ότι έχουμε κωδικό πελάτη 010101 0005, κωδικο χρήστη =0011111 και password = a11111 και θέλουμε να εκτυπώσουμε το voucher ΝΖ000000011GR μέσω elta_print.exe. Οπότε απο την γραμμή εντολών εκτελούμε (γράφουμε) elta_print.exe 010101 0011111 a11111 ΝΖ000000011GR)
3 Τα WS δεν επικοινωνούν με τον server της ELTA COURIE Βεβαιωθείτε ότι έχετε ρυθμίσει τα συστήματά σας ώστε να δέχεται traffic (εισερχόμενο/εξερχόμενο) απο την ip 212.205.47.226, πόρτα 9003

Απαιτείται να είναι ανοιχτή η πόρτα 9003 για TCP OUT συνδέσεις στο firewall. Θα πρέπει να ανοίξετε στο firewall την πόρτα 9003 προς την IP 212.205.47.226.
4 Κατα την εκτέλεση του ELTA_PRINT.EXE μου εμφανίζει error "Δεν έχει οριστεί IP στο ip.txt" 1. Βεβαιωθείτε ότι ο φάκελος έχει όλα τα απαραίτητα δικαιώματα
2. Βεβαιωθείτε ότι υπάρχει το αρχείo ip.txt στον φάκελο
3. Bεβαιωθείτε ότι η εκτέλεση του ELTA_PRINT.EXE γίνεται χωρίς να παρεμβάλεται κάποιος άλλος φάκελος ( δείτε το παραδειγμα )
5 Πότε εκτυπώνονται πολλαπλά ΣΥ.ΔΕ.ΤΑ. ? Γενικά, με μία εντολή δεν εκτυπώνονται πολλαπλά ΣΥ.ΔΕ.ΤΑ.Ο μόνος τρόπος για να γίνει αυτό είναι να κατασκευαστεί μηχανισμός ο οποίος θα καλεί αυτόματα την λίστα των προς εκτύπωση ΣΥ.ΔΕ.ΤΑ. (loop).
Παρ' όλα ταύτα υπάρχουν περιπτώσεις που με μία εντολή μπορούν να παραχθούν πολλαπλά ΣΥ.ΔΕ.ΤΑ. (με την χρήση του CREATEAWB.WSDL). Οι περιπτώσεις αυτές είναι:
- τα τεμάχια του ΣΥ.ΔΕ.ΤΑ. είναι άνω του ενός
- το ΣΥ.ΔΕ.ΤΑ να φέρει επιβάρυνση ειδικής διαχείρισης (κωδικός 2 στο πεδίο pel_sur_1)
- το ΣΥ.ΔΕ.ΤΑ. να φέρει επιβάρυνση αντικαταβολής με επιταγή (κωδικός 1 στο πεδίο pel_ant_poso1)
