Παρακολούθηση ιστορικού του αντικειμένου - Tracing



ΜΕΘΟΔΟΣ
ΟΝΟΜΑ ΜΕΘΟΔΟΥ ΠΕΡΙΓΡΑΦΗ
READ Το WS PELTT01.WSDL Επιστρέφει το ιστορικό του αντικειμένου



INPUT FIELDS
ΠΕΔΙΟ ΤΥΠΟΣ ΠΕΡΙΓΡΑΦΗ ΑΠΑΙΤΗΣΗ
WPEL_CODE Από ΕΛΤΑ Courier Κεφαλικός Κωδικός Αποστολέα (Υποχρεωτικό πεδίο) (έστω δίνεται κωδικός αποστολέα π.χ. 010101 0005 τότε WPEL_CODE= 010101, έστω δίνεται κωδικός αποστολέα π.χ. 123456 τότε WPEL_CODE= 123456 )
WPEL_USER Από ΕΛΤΑ Courier Κωδικός Χρήστη (Υποχρεωτικό πεδίο)
WPEL_PASS Από ΕΛΤΑ Courier Κωδικός Ασφαλείας (Υποχρεωτικό πεδίο)
WPEL_VG String 13 digit Αριθμός Αποστολής
WPEL_REF Char max 30 digit Αλφαριθμητικό πεδίο έως 30 χαρακτήρες
WPEL_FLAG Char 1 digit 1 = Αναζήτηση με Αριθμό Αποστολής
2 = Αναζήτηση με Reference_No
default value 1		 Αυστηρά μία από τις δύο τιμές
Clarification: Αν η αναζήτηση γίνει με το ΣΥ.ΔΕ.ΤΑ. θα πρέπει το wpel_flag να είναι 1,
αντίστοιχα αν η αναζήτηση γίνει με το Reference Number θα πρέπει το wpel_flag να είναι 2



OUTPUT FIELDS
ΠΕΔΙΟ ΤΥΠΟΣ ΠΕΡΙΓΡΑΦΗ ΑΠΑΙΤΗΣΗ
ST-FLAG Int 0 = Επιτυχής Εισαγωγή
1 έως 99 Error codes
ST-TITLE String Περιγραφή Λάθους αν ST-FLAG > 0


ΕΑΝ ΤΟ ΑΝΤΙΚΕΙΜΕΝΟ ΕΧΕΙ ΠΑΡΑΔΟΘΕΙ ΕΠΙΣΤΡΕΦΕΙ:
ΠΕΔΙΟ ΤΥΠΟΣ ΠΕΡΙΓΡΑΦΗ ΑΠΑΙΤΗΣΗ
POD_DATE Char 8 digit H ημερομηνία με format ΕΕΕΕΜΜΗΗ
POD_TIME Char 4 digit Η ώρα με format ΩΩΛΛ
POD_NAME String Όνομα παραλήπτη length=50


ΓΙΑ ΚAΘΕ ΓΡΑΜMH STATUS (WEB_STATUS) ΕΠΙΣΤΡΕΦΕΙ:
ΠΕΔΙΟ ΤΥΠΟΣ ΠΕΡΙΓΡΑΦΗ ΑΠΑΙΤΗΣΗ
WEB_DATE Char 8 digit H ημερομηνία με format ΕΕΕΕΜΜΗΗ
WEB_TIME Char 4 digit Η ώρα με format ΩΩΛΛ
WEB_STATION String Τίτλος Σταθμού που έκανε την κίνηση length=30
WEB_STATUS_TITLE String Περιγραφή του STATUS length=150


KAI ΣΑΝ ΤΕΛΕΥΤΑΙΑ ΠΛΗΡΟΦΟΡΙΑ ΕΠΙΣΤΡΕΦΕΙ:
ΠΕΔΙΟ ΤΥΠΟΣ ΠΕΡΙΓΡΑΦΗ ΑΠΑΙΤΗΣΗ
WEB_STATUS_COUNTER Int Συνολικός Αριθμός Κινήσεων Clarification: H ταξινόμηση είναι κατά φθίνουσα χρονολογική σειρά,
ταξινομώντας πρώτα την ημερομηνία και μετά την ώρα
