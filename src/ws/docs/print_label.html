Εκτύπωση αυτοκόλλητου ΣΥΔΕΤΑ - Label ( γίνεται κλίση του WS και επιστρέφεται pdf (σε μορφή base64))



ΜΕΘΟΔΟΣ
ΟΝΟΜΑ ΜΕΘΟΔΟΥ ΠΕΡΙΓΡΑΦΗ
READ Το WS PELB64VG.wsdl επιστρέφει pdf (σε μορφή base64) για την ετύπωση του label



INPUT FIELDS
ΠΕΔΙΟ ΤΥΠΟΣ ΠΕΡΙΓΡΑΦΗ ΑΠΑΙΤΗΣΗ
PEL_CODE Από ΕΛΤΑ Courier Κωδικός Αποστολέα (ολόκληρο - master + 6 κενα+ child - π.χ. 010101 0005) (Υποχρεωτικό πεδίο)
WPEL_USER Από ΕΛΤΑ Courier Κεφαλικός Κωδικός Χρήστη (Υποχρεωτικό πεδίο)
WPEL_PASS Από ΕΛΤΑ Courier Κωδικός Ασφαλείας (Υποχρεωτικό πεδίο)
WPEL_VG String 13 digit Αριθμός Αποστολής (Υποχρεωτικό πεδίο)
PAPER_SIZE String 0 = Label A4
1 = Label A6
Default Value 1		 Αυστηρά μία από τις δύο τιμές.

Clarification: Αν είναι επιθυμητό, το ΣΥΔΕΤΑ να εκτυπωθεί σε
1. Α4 τότε το paper_size=0
2. Α6 τότε το paper_size=1




OUTPUT FIELDS
ΠΕΔΙΟ ΤΥΠΟΣ ΠΕΡΙΓΡΑΦΗ
ST-FLAG Int 0 = Επιτυχής Εισαγωγή
1 έως 99 Error codes
Default Value 1
ST-TITLE String Περιγραφή Λάθους αν ST-FLAG > 0
Β64_STRING String BASE 64 string αν ST-FLAG = 0



Clarification: Παράδειγμα μετατροπής BASE 64 σε PDF (κώδικας σε c#) string file_name = @"c:\"+ VG_CODE + ".pdf";
Byte[] bytes_2 = Convert.FromBase64String(Β64_STRING);
File.WriteAllBytes(file_name, bytes_2);






Μορφή Αυτοκόλλητου ΣΥ.ΔΕ.ΤΑ (Α6) το οποίο επιστρέφει κατά την κλήση του Web Service (PELB64VG.wsdl ) όταν οριστεί το PAPER_SIZE = 1


Μορφή Αυτοκόλλητου ΣΥ.ΔΕ.ΤΑ (Α4) το οποίο επιστρέφει κατά την κλήση του Web Service (PELB64VG.wsdl ) όταν οριστεί το PAPER_SIZE = 0
