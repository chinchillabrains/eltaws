Ταυτοποίηση του Τ.Κ. με το κατάστημα εξυπηρέτησης



ΜΕΘΟΔΟΣ
ΟΝΟΜΑ ΜΕΘΟΔΟΥ ΠΕΡΙΓΡΑΦΗ
READ Το WS PELSTATION.wsdl επιστρέφει όλα τα στοιχεία του καταστήματος που εξυπηρετεί το συγκεκριμένο κατάστημα



INPUT FIELDS
ΠΕΔΙΟ ΤΥΠΟΣ ΠΕΡΙΓΡΑΦΗ
WEB_TK String 5 digit Ταχυδρομικός Κωδικός (Υποχρεωτικό πεδίο)



OUTPUT FIELDS
ΠΕΔΙΟ ΤΥΠΟΣ ΠΕΡΙΓΡΑΦΗ
web_code string Κωδικός Καταστήματος
web_company string Δίκτυο μεταφοράς
web_title string 'Ονομα Σταθμού Εξυπηρέτησης
web_address string Διεύθυνση Σταθμού
web_area string Περιοχή Σταθμού
web_wrario string Ωράριο Λειτουργίας Καταστήματος
web_tel1 string Τηλέφωνο Καταστήματος
web_fax string Τηλεομοιότυπο Καταστήματος
web_x string Χ συντεταγμένη
web_y string Υ συντεταγμένη
st_flag int Τύπος Καταστήματος
st_title string Τίτλος Σταθμού Εξυπηρέτησης (συνήθως είναι κενό)
